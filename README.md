# financial-audit
การตรวจสอบการเงิน (financial audit) เป็นการตรวจสอบการแสดงข้อมูลทางการตรวจสอบการเงินของสำนักงานการตรวจเงินแผ่นดิน โดยมีหลักเกณฑ์มาตรฐานการตรวจสอบการเงิน ขอบเขตของมาตรฐานการตรวจเงินแผ่นดิน การเงินของหน่วยรับตรวจว่าเป็นไปตาม แม่บทการรายงานทางการเงิน และอาจรวมถึงการตรวจสอบการปฏิบัติตามกฎหมาย
